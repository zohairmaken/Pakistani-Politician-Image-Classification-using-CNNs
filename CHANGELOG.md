# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-12
### Added
- Initial project structure with modular architecture.
- YAML-based configuration management.
- Multi-source image downloader (Google & Bing).
- Intelligent face extraction using MTCNN.
- Identity verification using Facenet512 embeddings.
- CNN training pipeline for ResNet50 and EfficientNetB0.
- Streamlit-based real-time monitor dashboard.
- Comprehensive evaluation reporting (Confusion Matrices, F1-Scores).

### Changed
- Refactored legacy scripts into the `src/` package.
- Improved documentation with a premium README and architecture diagrams.
